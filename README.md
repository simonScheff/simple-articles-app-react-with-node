# Blog App

## Getting Started 
On `my-blog` folder and on `be-my-blog-folder` run:
#### `npm i`

### Open 3 terminals:

### Start DB
on `/be-my-blog/` folder Run:
#### `mongod -dbpath ./mongo-db-data/`

### Start BE

go to `/be-my-blog/` run:
#### `npm start`

### Start FE

go to `/my-blog/` run:
#### `npm start`
